# 💼 Skill Salary Analysis & WFH vs Non-WFH Salary Comparison

This project presents a data analysis and visualization of job salaries, focusing on two main themes:

1. **Skill Salary Analysis**
2.![Persentase WFH](Screenshot%202026-06-08%20190710.png)

The project uses data from the following files located in the `/Resources` folder:

- `data_jobs_Salary_all.csv`
- `data_jobs_Salary_monthly.csv`
- Monthly data files: `jan.csv`, `feb.csv`, ..., `dec.csv`

These datasets include information such as:

- Job title
- Employment type
- Skill requirements
- Salary (annual & monthly)
- Remote (WFH) status
- Posting month

...
